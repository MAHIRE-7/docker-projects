# Todo App

A simple Node.js todo application without database (uses in-memory storage).

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open http://localhost:3000 in your browser

## Features

- Add new todos
- Mark todos as completed
- Delete todos
- All data stored in memory (resets on server restart)
